# 🛒 Simple Marketplace

Proyek **Simple Marketplace** adalah aplikasi sederhana untuk simulasi jual-beli produk secara online.  
Tujuan utama proyek ini adalah untuk belajar **GitHub workflow**, dokumentasi, dan pengelolaan kode.

---

## 📖 Deskripsi
Marketplace ini memungkinkan pengguna untuk:
- Melihat daftar produk
- Menambahkan produk ke keranjang
- Melakukan checkout sederhana

---
